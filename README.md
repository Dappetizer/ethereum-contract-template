# Ethereum Contract Template
A repo of template ethereum contracts.

| Contract | Description |
| ----------- | ----------- |
| BuyableERC721 | NFT contract with buyable mints. |
| CustomERC721 | Custom generic ERC721 implementation. |

## Build

```
truffle build
```

## Migrate

```
truffle migrate
```

## Test

```
truffle test
```
